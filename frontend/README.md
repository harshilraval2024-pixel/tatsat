# Frontend (Next.js)

Marketing site and staff dashboard UI for Tatsat NRGS.

- Install: `npm install`
- Dev: `npm run dev` → [http://localhost:3000](http://localhost:3000)

From the **repository root**, use `npm run dev` (delegates here).

Configure **`NEXT_PUBLIC_API_URL`** if the FastAPI server is not proxied via `/nrgs-api` (see `next.config.ts`).
